# Personal Portfolio
Personal Portfolio
Zechariah Sentz Portfolio

This project is a personal portfolio website focused on filmmaking, storytelling, and web development. The site showcases creative film projects, screenwriting interests, technical skills, and professional information through a responsive multi-page design.

The website was built using semantic HTML and mobile-first CSS design principles. Flex box is used for navigation and content cards, while CSS Grid is used for the overall page layout.

Pages:
- `index.html` — Homepage and biography
- `projects.html` — Film projects and creative work
- `resume.html` — Education, skills, and experience
- `contact.html` — Contact information and portfolio links
